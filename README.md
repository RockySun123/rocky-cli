# rocky-cli

![Npm 版本](https://img.shields.io/badge/rocky-cli_v0.0.1-green)

## 📖 简介

- ✅ 用于快速搭建前端项目的前端命令行工具

## 📕 模版

- 快速生成 Vue3 + Vite + Typescript + Pinia + Axios + Mock + Element-plus 模版

## 🎁 安装

```bash
npm install rocky-cli -g
```

## 🚴 使用

```bash
# 使用模版创建新项目
rocky create

# 执行项目文件夹名称病选择模版创建新项目
rocky create [projectName]

# 查看脚手架版本
rocky -v
rocky --version
```
