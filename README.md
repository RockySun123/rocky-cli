# rockys-cli

![Npm 版本](https://img.shields.io/badge/rockys-cli_v0.1.1-green)

## 📖 简介

- ✅ 用于快速搭建前端项目的前端命令行工具

## 📕 模版

- 快速生成 [Vue3 + Vite + Typescript + Pinia + Axios + Mock + Element-plus](https://gitee.com/sun-kelin/vue-infrastructure) 模版

## 🎁 安装

```bash
npm install rockys-cli -g
```

## 🚴 使用

```bash
# 使用模版创建新项目
rockys create

# 执行项目文件夹名称病选择模版创建新项目
rockys create [projectName]

#更新脚手架到最新版本
rockys update

# 查看脚手架版本
rockys -v
rockys --version
```
